# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, please report it privately so we can investigate and fix it promptly.

- **Contact email:** lethanhtrung.trungle@gmail.com
- **Please include:** steps to reproduce, affected area (page/API), expected vs. actual behavior, and any logs/screenshots if available.
- **Response target:** We aim to acknowledge reports within 5 business days and provide a remediation plan or ETA afterward.

## Scope

- This portfolio application (frontend, Next.js pages/components).
- Build/config files (Dockerfile, CI workflows).

## Out of Scope (examples)

- Self-XSS caused by pasting untrusted scripts in the browser console.
- Denial-of-service caused by automated load testing without prior agreement.

Thank you for helping keep this project secure.
